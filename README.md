# i3_typing_master_2026 ⌨️

A Java-based console application designed to test and improve typing speed and accuracy. This project was developed as a midterm assignment and features real-time performance tracking, including Words Per Minute (WPM), Characters Per Minute (CPM), and mistake calculations.

## 📂 Project Structure

* **`Main.java`**: The core entry point of the application. **This is currently the only file you need to compile and run** to experience the full, working program.
* **`Collaborator/`**: This directory contains work-in-progress modules (such as password reset functionality) being developed by team members. These features are currently independent and have not yet been integrated into the `Main.java` execution flow.

## ⚙️ Prerequisites

To compile and run this project locally, you will need:
* **Java Development Kit (JDK)**: Make sure you have JDK 8 or higher installed on your system. You can verify your installation by running `java -version` and `javac -version` in your terminal.
* **Git**: To clone the repository.

## 🚀 Setup and Execution

Follow these steps to run the typing master program on your local machine:

**1. Clone the repository**
Open your terminal or command prompt and run:
```bash
git clone [https://github.com/SetharTyKun/i3_typing_master_2026.git](https://github.com/SetharTyKun/i3_typing_master_2026.git)
